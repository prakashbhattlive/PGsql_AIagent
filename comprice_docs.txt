üìå Purpose
This document describes a cleaned, merged dataset of computer specifications and prices. It is designed for tabular regression tasks where the goal is to predict the price of a PC or laptop based on its hardware and configuration.
üìä Dataset Overview
- Size: ~100,000 rows
- Type: Tabular, structured
- Target column: price
- Use case: Regression modeling, feature engineering, benchmarking
üß† Modeling Objectives
LLMs or ML models using this dataset should be able to:
- Engineer features from categorical and numeric data
- Handle high-cardinality variables (e.g., brands, models)
- Build robust regression models (e.g., tree-based, boosting, deep learning)
- Efficiently manage large-scale tabular data
üìÅ Feature Categories
1. Device Information
- device_type: Laptop or Desktop
- brand, model
- release_year, os, form_factor
2. CPU Specifications
- cpu_brand, cpu_model, cpu_tier
- cpu_cores, cpu_threads
- cpu_base_ghz, cpu_boost_ghz
3. GPU Specifications
- gpu_brand, gpu_model, gpu_tier
- vram_gb
4. Memory & Storage
- ram_gb
- storage_type, storage_gb, storage_drive_count
5. Display Characteristics
- display_type: LED, OLED, Mini-LED, IPS, VA, QLED
- display_size_in, resolution, refresh_hz
6. Other Features
- battery_wh, charger_watts, psu_watts
- wifi, bluetooth
- weight_kg, warranty_months
7. Target
- price: Float (USD)
